## 1. 安装与配置

### 1.1 环境要求

**系统要求：**
- Python 3.8 或更高版本
- Windows / macOS / Linux

**最低配置：**
- 内存：4GB
- 硬盘：500MB

**推荐配置：**
- 内存：8GB
- 硬盘：1GB

### 1.2 安装步骤

#### 步骤1：克隆仓库

```bash
git clone https://github.com/LX1309244704/weiyuan.git
```

### 1.3 配置文件

配置文件会自动创建在用户目录下：

**配置文件位置：**
- Windows：`C:\Users\用户名\.weiyuan\config.json`
- macOS：`/Users/用户名/.weiyuan/config.json`
- Linux：`/home/用户名/.weiyuan/config.json`

```
~/.weiyuan/
├── config.json          # 主配置文件
├── feishu_tokens.json   # 飞书Token
├── wechat_cookies.json  # 微信Cookies
└── douyin_cookies.json  # 抖音Cookies
```

## 2. 快速开始

### 2.1 三种使用方式

微元 Weiyuan 提供三种使用方式：

| 方式 | 适用场景 | 特点 |
|------|----------|------|
| **CLI命令行** | 开发者、自动化脚本 | 灵活、可集成到CI/CD |
| **PC客户端** | 普通用户、日常操作 | 图形化界面、无需命令行 |
| **Web控制台** | 团队协作、远程管理 | 浏览器访问、实时监控 |

### 3.2 创建流程

#### 使用内置模板创建

```bash
# 查看可用模板
python -m rpa init --list-templates

# 使用模板创建流程
python -m rpa init 我的流程 -t wechat_send_message
```

## 6. 创建自动化流程

### 6.1 使用模板快速创建

```bash
# 1. 查看可用模板
python -m rpa init --list-templates

# 2. 使用模板创建
python -m rpa init 销售数据同步 -t daily_sales_report

# 3. 编辑流程
vim rpa/flows/销售数据同步.yaml
```

### 6.2 从零创建流程

创建一个新的YAML文件：

```yaml
name: 我的自动化流程
description: 示例流程
version: 1.0.0

trigger:
  type: manual  # 手动触发

steps:
  # 步骤1：获取数据
  - name: 获取飞书数据
    uses: lark/cli@1.0.0
    with:
      command: api
      service: base
      method: records/list
      params:
        app_token: "${{ env.FEISHU_APP_TOKEN }}"
    register: feishu_data

  # 步骤2：处理数据
  - name: 处理数据
    uses: data/process@1.0.0
    with:
      action: format
      data: "${{ steps.获取飞书数据.output.stdout }}"

  # 步骤3：发送通知
  - name: 发送微信通知
    uses: wechat/work@1.0.0
    with:
      action: send_text
      receiver: "运营部"
      content: "数据已同步完成！"
```

### 6.3 执行流程

```bash
python -m rpa run 我的自动化流程.yaml
```

## 7. 使用官方CLI

### 7.1 官方CLI命令类型

#### 1. 快捷命令（Shortcuts）

```bash
# 查看今日日程
lark-cli calendar +agenda

# 发送消息
lark-cli im +messages-send --chat-id "oc_xxx" --text "Hello"
```

#### 2. API命令（API Commands）

```bash
# 列出多维表格
lark-cli base tables/list --params '{"app_token":"xxx"}'

# 获取记录
lark-cli base records/list --params '{"app_token":"xxx","table_id":"xxx"}'
```

#### 3. 通用API（Generic API）

```bash
# 调用任意端点
lark-cli api GET /open-apis/calendar/v4/calendars
```

### 7.2 在流程中使用官方CLI

```yaml
steps:
  # 使用快捷命令
  - name: 获取日程
    uses: lark/cli@1.0.0
    with:
      command: shortcut
      service: calendar
      shortcut: +agenda

  # 使用API命令
  - name: 获取数据
    uses: lark/cli@1.0.0
    with:
      command: api
      service: base
      method: tables/list
      params:
        app_token: "${{ env.APP_TOKEN }}"
```

## 8. 常见问题

### Q1: 如何安装PC客户端依赖？

```bash
pip install -r requirements_gui.txt
```

### Q2: 如何配置飞书App ID和Secret？

1. 打开飞书开放平台：https://open.feishu.cn/
2. 创建应用获取App ID和Secret
3. 打开PC客户端 → 配置管理 → 飞书
4. 填入App ID和Secret
5. 点击"测试连接"验证

### Q3: 官方CLI如何登录？

```bash
lark-cli auth login --recommend
```

会自动打开浏览器，扫码登录即可。

### Q4: 如何查看执行日志？

```bash
# CLI方式执行时，日志直接输出到控制台

# PC客户端：流程管理 → 查看日志

# Web控制台：日志页面查看
```

### Q5: 如何设置定时任务？

在流程YAML中配置：

```yaml
trigger:
  type: schedule
  cron: "0 9 * * *"  # 每天早上9点
```
