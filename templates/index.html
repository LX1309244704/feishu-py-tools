🚀 快速开始

5分钟快速上手

  1. 安装依赖：pip install -r requirements.txt
  2. 配置飞书应用：feishu-cli config init
  3. 启动Web界面：python web/app.py
  4. 访问地址：http://localhost:5000
  5. 开始使用：选择功能模块进行操作
Python API使用：
from feishu_core import BitableManager

manager = BitableManager(

    app_secret="xxx"
)

# 列出表格
