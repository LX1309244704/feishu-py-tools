## 📋 安装使用

### 第1步：下载Skill包
```bash
# 从GitHub下载
wget https://github.com/LX1309244704/weiyuan/releases/download/v1.0.0/feishu-enterprise-builder.skill

# 或从虾评下载（消耗2虾米）
# https://xiaping.coze.site
```

### 第2步：解压安装
```bash
# 解压Skill包
unzip feishu-enterprise-builder.skill
cd feishu-enterprise-builder

# 安装依赖
pip install -r requirements.txt

# 安装工具
pip install -e .
```

### 第3步：配置环境变量
```bash
# 复制配置文件
cp .env.example .env

# 编辑配置
nano .env
```

### 第4步：授权配置
```bash
# 初始化
feishu-cli config init

# 授权
feishu-cli config auth
```

### 第5步：开始使用
```bash
# 创建表格
feishu-cli base create --name "客户管理" --template crm

# AI辅助录入
feishu-cli base ai-fill --prompt "添加10个客户信息"

# 生成仪表盘
feishu-cli viz dashboard --table-id xxx
```

### 场景1：快速搭建客户管理系统

**效果**：

- ✅ 20个客户信息
- ✅ 实时数据仪表盘

---

### 场景2：自动化销售跟进
**时间**：10分钟
**命令**：
```bash
# 创建工作流
cat > sales_followup.yaml << EOF
name: 销售跟进自动化
trigger:
  type: data_change
  field: 状态
  value: 新建

steps:
  - id: step1
    name: 创建跟进任务
    api: task.create
    params:
      title: "跟进客户：${客户名称}"
      due: "+3d"

  - id: step2
    name: 发送通知
    api: message.send
    params:
      content: "新客户需要跟进"
EOF

# 运行工作流
feishu-cli workflow run --config sales_followup.yaml
```

**效果**：
- ✅ 自动创建跟进任务
- ✅ 自动发送通知
- ✅ 自动记录跟进状态
