# 🖥️ RPA工具PC客户端使用指南
基于PySide6开发的跨平台PC客户端，支持Windows/macOS/Linux。

---

## ✨ 客户端特性
- 🎨 可视化界面，操作简单直观
- 📝 内置YAML编辑器，语法高亮、实时验证
- ▶️ 一键执行流程，实时查看执行日志
- 📊 首页仪表盘，统计概览一目了然
- 📋 执行历史记录，支持重新执行
- 📦 模板市场，常用模板一键使用
- 🔌 插件中心，管理所有插件
- ⚙️ 系统设置，飞书授权、个性化配置

---

## 🚀 快速启动

### 方式1：直接运行源码（推荐开发者）
1. 安装依赖
```bash
# 安装基础依赖
pip install -r requirements.txt

# 安装RPA工具依赖
pip install -r requirements_rpa.txt

# 安装GUI依赖
pip install -r requirements_gui.txt
```

2. 启动客户端
```bash
python start_gui.py
```

### 方式2：打包成可执行文件
1. 执行打包脚本
```bash
# Linux/macOS
chmod +x build_gui.sh
./build_gui.sh

# Windows
pip install pyinstaller
pyinstaller --onefile --windowed --name "RPA工具" start_gui.py
```

2. 打包完成后，可执行文件在`dist/`目录下，双击即可运行。

## 📖 界面功能介绍

### 1. 首页 Dashboard
- **统计卡片**：总流程数、今日执行数、成功率、平均耗时
- **最近执行记录**：查看最近的流程执行情况，一键重新执行
- **常用模板**：快速使用内置的常用流程模板

### 2. 流程管理
- **流程列表**：显示本地所有流程文件
- **YAML编辑器**：语法高亮、自动补全
- **实时验证**：一键验证流程格式是否正确
- **执行日志**：实时查看流程执行的输出日志
- **快捷操作**：新建、打开、保存、执行、验证

### 3. 执行记录
- **历史记录列表**：所有流程的执行历史
- **搜索筛选**：按流程名称、执行状态筛选
- **详情查看**：查看步骤执行详情、耗时、日志
- **重新执行**：失败流程一键重试
- **导出功能**：导出执行记录报表

### 4. 模板市场
- **内置模板**：库存预警、销售报表、入职流程等常用模板
- **分类筛选**：按功能分类查看模板
- **一键使用**：点击直接导入模板到本地编辑

### 5. 插件中心
- **已安装插件**：查看所有已注册的插件
- **插件详情**：查看插件功能、版本、作者信息
- **插件市场**：（开发中）在线安装更多插件

### 6. 设置
- **飞书配置**：配置默认的飞书AppID和AppSecret
- **通用设置**：主题、语言、开机启动等
- **路径配置**：流程和日志的默认保存路径
- **配置保存**：自动持久化到本地

## 🎯 使用流程示例
### 步骤1：创建流程
1. 点击首页"新建流程"或在流程管理页点击"新建"
2. 输入流程名称和描述
3. 在YAML编辑器中编写流程定义
4. 点击"验证"按钮检查格式是否正确

### 步骤2：执行流程
1. 点击"执行"按钮
2. 切换到"执行日志"标签页查看实时输出
3. 执行完成后查看执行结果和耗时

### 步骤3：查看记录
1. 切换到"执行记录"页面
2. 查看所有历史执行记录
3. 可以筛选、重新执行、查看详情

## 🔧 常见问题

### Q: 启动时报错找不到PySide6？
A: 请确保安装了GUI依赖：`pip install -r requirements_gui.txt`

### Q: 流程执行失败？
A: 1. 先点击"验证"按钮检查流程格式是否正确
2. 检查飞书授权信息是否正确
3. 查看执行日志中的具体错误信息

### Q: 如何打包成Windows exe？
A: 在Windows环境下执行build_gui.sh脚本，或者直接执行：
```bash
pyinstaller --onefile --windowed --name "RPA工具" start_gui.py
```
打包后的文件在dist目录下。

### Q: 如何添加自定义图标？
A: 修改build_gui.sh中的--icon参数，指定你的ico图标路径即可。
