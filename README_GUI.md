# 🖥️ 微元 PC客户端使用指南

## 🚀 快速启动

### 1. 安装依赖

```bash
cd weiyuan

# 安装GUI依赖
pip install -r requirements_gui.txt
```

### 2. 启动客户端

```bash
python start_gui.py
```

启动后会看到可视化窗口，所有操作都可以通过界面完成。

---

## 🎯 核心功能

### 1️⃣ 配置管理（无需命令行！）

**所有API密钥、Token都在GUI中配置：**

1. 点击顶部菜单 **"工具"** → **"配置管理"**（或按 `Ctrl+Shift+C`）
2. 在配置对话框中填写各平台参数：
   - 飞书：App ID、App Secret
   - 企业微信：Corp ID、Agent ID、Corp Secret
   - 公众号：App ID、App Secret
   - AI：OpenAI/Claude/通义千问 API Key
3. 点击 **"保存配置"** 即可

配置会自动保存到 `~/.weiyuan/config.json`，流程文件中使用变量引用即可。

### 2️⃣ 首页仪表盘

- 📊 统计卡片：总流程数、今日执行数、成功率、平均耗时
- 📋 最近执行记录：查看最近的流程执行情况
- 📦 常用模板：快速使用内置模板

### 3️⃣ 流程管理

- 📝 流程列表：查看本地所有流程文件
- ✏️ YAML编辑器：带语法高亮的编辑器
- ▶️ 执行流程：一键执行，实时查看日志
- ✅ 验证格式：检查YAML格式是否正确

### 4️⃣ 执行记录

- ⏱️ 历史记录：所有流程的执行历史
- 🔍 筛选查询：按名称、状态筛选
- 🔄 重新执行：失败流程一键重试

### 5️⃣ 模板市场

- 📦 内置模板：库存预警、销售报表、微信发消息等
- ⚡ 一键使用：点击直接导入到本地编辑

### 6️⃣ 插件中心

- 🔌 查看已安装插件
- 📖 插件功能说明

### 7️⃣ 设置页面

- ⚙️ 快速打开配置管理器
- 📊 配置状态概览
- ℹ️ 项目信息

## 💡 使用流程示例

### 示例：配置飞书并执行流程

1. **配置飞书参数**
   - 菜单：工具 → 配置管理
   - 切换到"飞书"标签页
   - 填写 App ID 和 App Secret
   - 点击"保存配置"

2. **创建流程**
   - 点击首页"新建流程"或菜单：文件 → 新建流程
   - 填写流程名称
   - 在编辑器中编写YAML流程

3. **执行流程**
   - 切换到"流程管理"页面
   - 选择要执行的流程
   - 点击"执行"按钮
   - 在"执行日志"标签页查看实时输出

## 📁 配置文件位置

所有配置都保存在：

```
~/.weiyuan/
├── config.json          # 主要配置文件
└── logs/                # 日志目录
```

**注意**：配置文件包含敏感信息（API密钥），请妥善保管，不要上传到GitHub。

## 🔧 打包成可执行文件

### Windows 打包

```bash
pip install pyinstaller

pyinstaller --onefile --windowed \
    --name "微元Weiyuan" \
    --icon rpa/gui/resources/icons/app.png \
    start_gui.py
```

打包完成后在 `dist/` 目录下会生成 `微元Weiyuan.exe`，双击即可运行。

### macOS 打包

```bash
pip install pyinstaller

pyinstaller --onefile --windowed \
    --name "微元Weiyuan" \
    start_gui.py
```

生成 `.app` 文件后，拖到应用程序文件夹即可。

## ❓ 常见问题

### Q: 启动时提示找不到PySide6？
A: 请安装GUI依赖：`pip install PySide6`

### Q: 配置文件保存在哪里？
A: 保存在用户主目录下的 `.weiyuan/config.json`
- Windows: `C:\Users\用户名\.weiyuan\config.json`
- macOS/Linux: `~/.weiyuan/config.json`

### Q: 如何备份配置？
A: 复制 `~/.weiyuan/config.json` 文件即可。

### Q: 如何重置配置？
A: 在配置管理器中点击"重置默认"按钮，或删除 `~/.weiyuan/config.json` 文件。

### Q: 流程文件保存在哪里？
A: 默认保存在配置中设置的"流程目录"，也可以在"通用"配置页修改。
